Supabase not configured

Phase 2 needs a Supabase project for realtime sync. Set it up once:

  1. Go to supabase.com, sign in, and create a new project (free tier is fine).
  2. From Project Settings → API, copy the Project URL and anon public key.
  3. Open shared.js and replace the placeholder values of SUPABASE_URL and SUPABASE_ANON_KEY.
  4. Reload this page.

The anon key is public by design — safe to commit.
